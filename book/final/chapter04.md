
## アルゴリズムの初速重視

あきらパパ「Xのアルゴリズムは『初速』を非常に重視するんだ。」

### 初速の重要性

```
[初速によるその後の展開]

最初の30分のエンゲージメント：
- 高い（10%以上）→ 拡散フェーズへ
- 中程度（5-10%）→ 限定的な拡散
- 低い（5%未満）→ ほぼ拡散しない

拡散の仕組み：
1. 初期露出（フォロワーの10-20%）
2. エンゲージメント測定
3. スコア計算
4. 拡散判定

   → 低スコア：露出終了
```
